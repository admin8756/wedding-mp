## 说明：

这个是我结婚时候自己开发的小程序，因为云开发已经到期了。所以视频和弹幕都没有了。

如果你要拿来用。那么请自己改一改。主要是背景图片。还有一些参数。目前没有抽离到config文件里。

请帖的样式你应该要重新设计一下。

如果你觉得有用。请给本项目点个赞。
